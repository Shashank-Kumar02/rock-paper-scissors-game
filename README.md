# 🎮 How to Play
<br/>
- Click on **Rock**, **Paper**, or **Scissors**.
  <br/>
- The computer will randomly choose its move.
  <br/>
- The result (win/lose/draw) will be displayed along with updated scores.
  <br/>
- Game continues until you refresh the page.
